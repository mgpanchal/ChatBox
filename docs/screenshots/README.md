# Screenshots

Drop product screenshots here so they render in the project README.

Expected filenames (referenced from the root [README.md](../../README.md)):

| File              | Capture                                       | Suggested size |
| ----------------- | --------------------------------------------- | -------------- |
| `web-chat.png`    | Web client — open chat thread                 | 1440×900       |
| `web-admin.png`   | Web client — admin dashboard (users / audit)  | 1440×900       |
| `mobile-chat.png` | Mobile — open chat thread                     | 1170×2532 (iPhone 14 Pro) or 1080×2400 (Pixel) |
| `mobile-list.png` | Mobile — chats list / sidebar                 | 1170×2532 or 1080×2400 |

Tips:

- Use real-looking but **anonymised** data (no real employee names, phone
  numbers, or message content).
- Crop the OS chrome (status bar, navigation bar) only if it makes the shot
  cleaner — otherwise leave it for context.
- Prefer light theme for consistency.
- Keep file sizes under ~500 KB each (use `pngquant` or `oxipng`).
